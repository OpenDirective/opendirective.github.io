Using Microsoft Azure in a Serverless mode

Serverless Sessions from Microsoft Connect(); // 2016

By steve@opendirective.com on November 18, 2016 • ( Leave a comment )

Microsoft have certainly hit on a winning formula with the annual Connect developer event. This year,  2016, was even better than last year, with new technologies, some of which where promised in 2015 bore  fruits. The first day of Keynotes was full of announcements of cool new tools and services. Day 2 was a deep dive into specific product features and how to choose athem. Day 3 promises more of the same.

Here’s a selection of videos from the full on demand collection. They’ve been picked for their relevance to distributed architectures using Azure Serverless.

  • Build and Integrate Applications Faster Using Azure App Service – Andrew Hall, Jon Fancey, Seth Juarez, Yochay Kiritay highlight how App Service features including LogicApps and Functions enable rapid development of cloud hosted apps.
  • Build Microservice Applications with Microsoft Azure – an excellent deep dive discussion by  Chris Anderson, Mark Fussell, Seth Juarez & Ross Gardler on the options for developing microservices using Functions, Service Fabric and Container Service.
  • DocumentDB: overview and offline development experience – Kirill Gavrylyuk on the Azure NoSQL database
  • Application Insights Diagnostic Preview – improved Application Map and Application Insights Profiler from Application Insights which can be used with Functions.

And here’s a few extras of general interest

  • Improvements to Bash on Windows and the Windows Console – Rich Turner introduces this much requested feature for developers along with the long overdue improvements to the terminal emulation.
  • What’s new in Visual Studio Code – Wade Anderson introduce some of the best new Code features.
  • Dive into the new world of Windows Server and Hyper-V containers – Taylor Brown previews a different type of container.
